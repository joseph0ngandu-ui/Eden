OVERVIEW
================================================================================

Eden is an advanced algorithmic trading bot designed to execute profitable 
trades across multiple financial instruments using optimized parameter-based 
strategies. The system employs machine learning-assisted optimization and 
technical analysis to identify high-probability trading opportunities with 
managed risk exposure.

The bot is engineered specifically for micro-account trading ($100 capital) 
and has been optimized through 50 iterations of parameter tuning to achieve 
exceptional risk-adjusted returns.

CORE FUNCTIONS
================================================================================

1. DATA INGESTION & MANAGEMENT
   • Loads OHLCV (Open, High, Low, Close, Volume) data from MT5 feeds
   • Supports 10 trading instruments: VIX75, VIX100, VIX50, VIX25, Crash1000,
     Crash500, Boom1000, Boom500, StepIndex, XAUUSD
   • Multiple timeframes: M1 (1-minute), M5, M15, 1H, 4H
   • Real-time data updates with fallback caching mechanisms
   • CSV data format with timestamp normalization

2. TECHNICAL ANALYSIS & INDICATORS
   • Simple Moving Averages (SMA) - multiple periods
   • Exponential Moving Averages (EMA) - trend confirmation
   • Relative Strength Index (RSI) - momentum & overbought/oversold detection
   • Average True Range (ATR) - volatility measurement
   • Bollinger Bands - dynamic support/resistance zones
   • MACD - trend-following momentum indicator
   • Volume analysis - trade confirmation
   • Higher Timeframe (HTF) Bias - multi-timeframe confluences

3. SIGNAL GENERATION & STRATEGY EXECUTION
   Implemented Strategies:
   
   BASE STRATEGIES (7):
   • Breakout Strategy - breakout on 10-period highs/lows
   • EMA Crossover with HTF Filter - EMA12/26 cross with SMA50 confirmation
   • Volume Momentum - RSI + volume spike confirmation
   • RSI Extreme - overbought/oversold reversals (Period 14, Threshold 30)
   • MACD Crossover - trend following with MACD signal line
   • Bollinger Bands + RSI - mean reversion with RSI confirmation
   • Confluence Multi-Indicator - 3+ aligned signals required
   
   ICT STRATEGIES (4):
   • Supply/Demand Zones - price action reversals at zone boundaries
   • Break of Structure (BOS) - trend reversals on recent extremes
   • Equal Highs/Lows - reversal signals on repeated price levels
   • Displacement Strategy - large ATR moves with trend confirmation
   
   ML-GENERATED STRATEGIES (5):
   • Mean Reversion - Z-score based overbought/oversold
   • Momentum Acceleration - second derivative momentum
   • Volatility Expansion - elevated volatility breakout detection
   • Entropy-Based Trading - market randomness detection
   • Pattern Recognition - higher lows/lower highs trend continuation

4. BACKTESTING ENGINE
   • Simulates trades on historical data with precision
   • Calculates accurate entry/exit points
   • Tracks individual trade PnL
   • Computes maximum drawdown per trade
   • Calculates win rate and profit factor
   • Supports $100 micro-account simulation
   • Handles both long and short positions
   • Position tracking with peak equity monitoring

5. PARAMETER OPTIMIZATION ENGINE
   • Tests multiple strategy parameters across 50 iterations
   • Generates 174 unique parameter combinations
   • Scores configurations by: Return / (Drawdown + 1)
   • Automatic best configuration selection
   • Per-instrument performance tracking
   • Parallel execution for speed (6 concurrent workers)
   • Random parameter sampling to avoid overfitting
   • Results archived for reproducibility

6. PORTFOLIO ANALYSIS & REPORTING
   • Per-instrument return calculations
   • Average portfolio return aggregation
   • Win rate analysis (% of profitable trades)
   • Profit factor computation (wins/losses ratio)
   • Maximum drawdown tracking
   • Trade count statistics
   • JSON output for system integration
   • Visual performance summaries

7. RISK MANAGEMENT (Built-in)
   • Position sizing: 1 unit = $1 risk per trade
   • Drawdown monitoring and alerts
   • Win/loss tracking
   • Capital preservation logic
   • Individual instrument performance limits
   • Emergency stops on negative instruments

Eden is a sophisticated automated trading system that combines technical 
analysis, mathematical optimization, and backtesting to generate trading 
signals. The system has demonstrated exceptional optimization results,

algorithmic trading system. Without these, it carries significant financial 
risk.
